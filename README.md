# NetSci4MissingPeople

## Data collection
Data was stored in a SQL database (PostgreSQL) collected by the Myosotis System https://github.com/raulsenaferreira/Projeto-Myosotis

This database contains 1794 registers about Brazilian missing persons from 7 different datasources containing 25 attributes including geo coordinates.

A extract of this data can be downloaded at https://zenodo.org/record/283320#.Y8v_hnbMJPY

## Dependencies
Numpy, Networkx
